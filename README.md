# Joke & Quote App

A simple, modern React application that fetches and displays random jokes and quotes.  
Users can toggle between jokes and quotes, fetch new content, and view structured content in a clean card layout.

## Features

- Fetches **random jokes** from [JokeAPI](https://v2.jokeapi.dev/)
- Fetches **random quotes** from [Quotable API](https://api.quotable.io/)
- Multi-line content display:
  - Jokes: **setup** and **punchline** on separate lines
  - Quotes: **content** and **author** on separate lines
- Modern, responsive UI:
  - Gradient buttons with hover effects
  - Rounded, centered cards
  - Mobile-friendly layout
- Loading state management
- Easy to extend for additional content sources
